# 蓝牙接收机手机 App 需求拆解（按页面）

> 来源：`蓝牙接收机手机app预研-PRD-20260110.pdf`（V1.1，2026-01-10）  
> 说明：按“用户流程 + 页面”方式拆解，供产品、设计、开发、测试共用。

## 1. 端到端流程（主链路）

1. 启动 App（图标页）
2. 播放启动动画（约 3s）
3. 进入开始页，判断手机蓝牙状态
4. 蓝牙未开则引导开启；蓝牙已开则可直接进入后续操作
5. 用户选择：
- 去配对（新设备配对）
- 已配对设备列表（连接/切换/删除）
- 开始（进入控制页）
- 设置（进入配置体系）
6. 控制页完成操控（方向、油门、车灯、陀螺仪、微调、提示）
7. 设置页可进入子页：通道设置、失控保护、履带混控、报警提示、固件升级、帮助中心

## 2. 页面拆解

## 页面1-2：文档信息页（封面/修订记录）

- 用途：文档管理信息，无 App 交互需求。
- 关键内容：版本 V1.0 / V1.1，V1.1 增加功能与页面优化。

## 页面3：启动与开始页

- 页面目标：完成冷启动、蓝牙可用性检查、主入口分流。
- 入口：App 打开。
- 展示要求：
- App 图标与名称（Flysky-Smart Car / 富斯智能遥控车）。
- 启动动画约 3s 后进入开始页。
- 开始页显示电压与信号（已连接时），未连接显示默认态。
- 关键交互：
- 检测蓝牙开关。
- 未开启：弹窗引导用户开启蓝牙。
- 已开启：可进入“已配对设备列表”或“去配对”。
- 按钮：`设置`、`开始`。
- 自动逻辑：
- 启动后在蓝牙开启状态下，自动搜索已配对设备在线状态。
- 搜索到在线设备时弹窗询问是否连接。
- 可选择“不再提示”，本次会话停止该提示。
- 弹窗：开启蓝牙、已连接设备发现、切换连接确认等（PRD 弹窗1~4）。

## 页面4：已配对设备列表

- 页面目标：管理历史配对设备，完成连接切换与删除。
- 入口：开始页弹窗“已配对设备列表”。
- 展示要求：
- 列表区分在线/离线设备，按最近使用时间排序。
- 最多展示 10 条；超过后淘汰最旧记录。
- 设备状态：`正在使用`、`正在连接`、空白（普通在线）、灰态（离线不可点）。
- 无设备时展示空态。
- 关键交互：
- 点击可连接设备：触发连接流程。
- 若当前有正在使用设备，切换前需二次确认（安全提示）。
- 支持删除设备；若删当前正在使用设备需强确认。
- 连接超时（30s）失败给出失败弹窗。
- 自动搜索：
- 进入页默认搜索 30s，之后切到未搜索状态。
- 支持“刷新”重新进入搜索。
- 约束：
- App 同时只控制 1 台接收机。
- 不识别非本 App 支持的蓝牙设备。

## 页面5：去配对

- 页面目标：搜索并配对新接收机。
- 入口：
- 启动流程中蓝牙可用后自动进入。
- 或开始页点击“去配对”进入。
- 展示要求：
- 配对中动态效果。
- 文案提示接收机需处于蓝牙模式（LED 快闪）。
- 关键交互与逻辑：
- 进入页后不主动连接旧设备，优先搜索可配对新设备。
- 搜索到匹配设备后执行配对，成功提示并返回开始页。
- 未搜索到则循环搜索。
- 若满足“已配对在线设备搜索条件”，可弹窗询问是否直接连接旧设备。
- 性能指标：
- 已对码设备（5m 内）搜索连接 < 200ms。
- 新设备（5m 内）搜索配对连接 < 300ms。

## 页面6-7：控制页（1/2, 2/2）

- 页面目标：完成车辆实时操控与反馈显示。
- 入口：开始页点击“开始”。
- 展示区块：
- 顶部信息栏：模型电量、信号强度（未连接时不显示）。
- 左/右控制区：方向与油门（受“手型设置”影响左右映射）。
- 车灯区：大灯、警示灯、刹车灯、倒车灯、转向灯（按通道配置显示）。
- 微调区：范围 ±50，精度 1（2us）。
- 低电压/低信号提示区。
- 音乐与音效开关。
- 关键交互：
- 方向控制：按触摸位移投影计算比例，越界回中。
- 油门控制：上滑/下滑区分前进与后退，松手回中。
- 固定位置/隐藏可变位置两种操控模式。
- 车灯：大灯/警示灯可点切换，刹车/倒车/转向灯按混控结果只读显示。
- 油门挡位：支持挡位选择（互斥）。
- 微调：支持单击与长按连调。
- 陀螺仪：
- 支持激活/关闭。
- 在“all”模式下，前后翻转影响油门、左右翻转影响方向。
- 角度区间对应油门/刹车变化（按 PRD 角度阈值执行）。
- 联动规则：
- 低电压、低信号提示由“报警和提示”配置驱动。

## 页面8：设置（总入口）

- 页面目标：集中管理控制参数、报警、升级、帮助。
- 入口：开始页/控制页进入设置。
- 功能项：
- 手型设置（左手油门 / 右手油门，默认右手油门）。
- 操控模式（固定位置 / 隐藏可变位置，默认固定）。
- 陀螺仪设置（方向 / all 等模式，互斥）。
- 通道设置。
- 失控保护设置。
- 履带混控。
- 设置接收机退出蓝牙模式。
- 报警和提示。
- 背景音乐设置（默认音乐 / 本地音乐）。
- 固件升级。
- 帮助中心。
- 特殊逻辑：
- “退出蓝牙模式”需连接设备后才能执行，执行前弹风险确认。
- 背景音乐本地文件仅支持 `.mp3`、`.wav`。
- 固件升级依赖网络与设备连接状态。

## 页面9：通道设置

- 页面目标：配置各通道映射、范围、反向与辅助功能。
- 入口：设置 -> 通道设置。
- 展示要求：
- CH1（方向）、CH2（油门）固定展示。
- CH3 起按设备支持通道数动态展示（当前适配至 CH4）。
- 行程参数范围：-100% ~ +100%，映射 1000us ~ 2000us，精度 1%。
- 反向复选框默认未选。
- 进度条展示行程与中立位置。
- 关键交互：
- 数值输入带校验（仅数字、长度限制、越界修正、空值回默认）。
- 辅助通道功能通过下拉弹窗选择（如无/大灯/警示灯/挡位控制/陀螺仪等）。
- 车灯逻辑参数可配置开关对应输出值。

## 页面10：失控保护设置

- 页面目标：配置失控后的通道输出策略。
- 入口：设置 -> 失控保护设置。
- 展示要求：
- 每通道可选 `固定值` / `保持`（默认固定值）。
- 选固定值时可配置通道值，默认 1500us，范围 1000us~2000us，精度 1us。
- 关键交互：
- 输入校验规则与通道设置一致。
- `TEST` 按键：
- 未按下 -> 断开控制信号，验证失控保护效果。
- 按下 -> 恢复正常控制。
- 离开页面时自动恢复未按下态。

## 页面11：履带混控

- 页面目标：配置履带车方向/油门混控比例。
- 入口：设置 -> 履带混控。
- 展示要求：
- 固定展示履带车俯视图。
- 配置项含百分比参数与结果进度条。
- 关键交互：
- 参数输入支持激活编辑、键盘输入、校验修正。
- 键盘显示时按钮区不可点击；点击其它区域可取消激活。
- 进度条为只读展示。

## 页面12：报警和提示

- 页面目标：配置低电压、低信号、断连/重连提示策略。
- 入口：设置 -> 报警和提示。
- 低电压报警：
- 开关控制整组显示。
- 可选电池类型（如 2S 等），联动默认电压参数。
- 参数：最低电压、满电电压、报警电量百分比。
- 报警方式：语音/振动（可勾选）。
- 触发频率：持续低压时每 30s 提醒。
- 低信号报警：
- 阈值范围 0~100，默认 30。
- 报警方式：语音/振动。
- 触发频率：持续低信号时每 10s 提醒。
- 断连/连上提示：
- 支持语音/振动开关。
- 当可控接收机断开或重新连上时提示。

## 页面13：固件升级

- 页面目标：在线读取固件列表并执行升级。
- 入口：设置 -> 固件升级。
- 展示要求：
- 固件列表来源于配置网址。
- 无可用文件时显示空态文案。
- 关键交互与流程：
- 点击固件后，先判断与当前版本关系。
- 当前已最新：提示“是否重刷”。
- 选择旧版本：提示“是否降级”。
- 确认后进入下载 + 升级流程，展示进度窗。
- 成功：弹成功提示（3s 自动消失/可手动关闭）。
- 失败：弹失败提示。
- 超时：任意环节卡住超过 1 分钟，退出并提示失败。

## 页面14：帮助中心

- 页面目标：查看在线说明书。
- 入口：设置 -> 帮助中心。
- 展示要求：
- 列表展示说明书条目（来源于网址）。
- 支持空列表状态。
- 关键交互：
- 点击条目打开 PDF 阅读页。
- 阅读页全屏展示，支持缩放与滚动。
- 缩放范围：最小为横向完整显示，最大 10 倍。
- 有关闭按钮可返回帮助中心。
- 异常：网络异常时提示弹窗（约 3s 自动消失，触摸可关闭）。

## 3. 全局规则汇总

- 连接规则：同一时刻仅允许控制 1 台接收机。
- 搜索规则：启动后自动搜索已配对在线设备；配对页优先搜索新设备。
- 超时规则：
- 列表连接超时约 30s 判失败。
- 固件升级任一环节超时 > 1min 判失败。
- 提示规则：
- 部分弹窗 3s 自动消失并支持点击关闭。
- 高风险操作（切换连接、退出蓝牙模式等）需二次确认。
- 依赖项：
- 音乐文件、部分 UI 细节、灯控信号映射由后续资料补齐。
